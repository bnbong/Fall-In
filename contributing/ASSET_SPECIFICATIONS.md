# 그래픽 에셋 규격 가이드 (Asset Specifications)

이 문서는 "헤쳐 모여! (Fall In!)" 게임에 사용되는 그래픽 에셋의 규격을 정의합니다.

---

## 에셋 폴더 구조

| 에셋 종류 | 폴더 위치 | 파일명 규칙 |
|----------|----------|------------|
| 대대원카드 초상화 | `assets/images/characters/portraits/` | `portrait_{id}.png` |
| 면담 상반신 | `assets/images/characters/soldiers/` | `soldier_{id}.png` |
| 일반 mob 초상화 | `assets/images/characters/mobs/` | `portrait_mob_danger_{level}_{variant}.png` |
| 지휘관 캐릭터 | `assets/images/characters/commander/` | `commander.png` |
| 배경 이미지 | `assets/images/ui/backgrounds/` | `{scene}_background.png` |
| UI 아이콘 | `assets/images/ui/icons/` | `icon_{name}.png` |
| 카드 베이스 | `assets/images/cards/` | `battalion_card_base_single.png` |

## 대대원카드 (Battalion Card)

```
┌─────────────────────────────────────┐
│           150px x 253px              │
├─────────────────────────────────────┤
│                                      │
│    ┌─────┐  ← 카드번호 원 (28px Ø)   │
│    │ 번호 │    5.5% from top         │
│    └─────┘                           │
│                                      │
│      ┌─────────────┐                 │
│      │             │                 │
│      │  초상화 영역  │ ← 90px 지름     │
│      │  (원형 마스크) │   30% from top │
│      │             │                 │
│      └─────────────┘                 │
│                                      │
│        ─── 이름 ───    ← 55%         │
│        ─── 계급 ───    ← 66%         │
│        ─── 소속 ───    ← 76%         │
│        ─── 위험도 ───   ← 86%        │
│                                      │
└─────────────────────────────────────┘
```

### 카드 베이스

- **크기**: 150px × 253px
- **형식**: PNG (투명 배경)
- **파일**: `battalion_card_base_single.png`

### 카드 초상화

- **크기**: 90px × 90px (원형 마스크 적용)
- **위치**: 카드 중앙, 상단 30%
- **형식**: PNG (투명 배경, 정사각형)

## 병사 상반신 이미지 (Soldier Bust)

면담 장면에서 사용되는 병사 상반신 이미지입니다.

```
┌───────────────────────────────────────┐
│         600px x 700px (권장)           │
│                                        │
│          ┌─────────────┐              │
│          │    머리     │              │
│          │             │              │
│          │    상체     │ ← 중앙 정렬   │
│          │             │              │
│          │    허리     │              │
│          └─────────────┘              │
│                  ↑                     │
│            하단 = 허리선               │
└───────────────────────────────────────┘
                  │
            앵커 포인트 (테이블 가장자리)
```

### 규격

- **소스 크기**: 600px × 700px (권장)
- **표시 크기**: 400px × 480px
- **앵커 Y**: 465px (테이블 가장자리)
- **형식**: PNG (투명 배경)
- **파일명**: `soldier_{id}.png` (예: `soldier_1.png`, `soldier_55.png`)

### 중요 사항

- 캐릭터는 이미지 중앙에 배치
- 하단이 허리 수준에서 잘리도록 구성
- 하단 고정 확대 애니메이션 (0.3x → 1.0x)

## 지휘관 캐릭터 (Commander)

게임 화면 좌측에 표시되는 지휘관 캐릭터입니다.

### 규격

- **크기**: 250px × 680px
- **위치**: X=100, Y=520 (하반신이 화면 밖으로 잘림)
- **형식**: PNG (투명 배경)
- **파일**: `commander.png`

## 배경 이미지 (Backgrounds)

### 규격

- **크기**: 1280px × 720px (화면 해상도)
- **형식**: PNG 또는 JPG
- **파일**: `{scene}_background.png`

### 필요한 배경

| 파일명 | 설명 |
|--------|------|
| `title_background.png` | 타이틀 화면 |
| `main_game_background.png` | 메인 게임 화면 |

## 냉동식품 아이콘 (Frozen Food)

면담 장면에서 테이블 위에 표시되는 BX 냉동식품 아이콘입니다.

### 규격

- **크기**: 40px × 40px (기본값, config에서 조정 가능)
- **형식**: PNG (투명 배경)
- **위치**: 테이블 중앙 X=500, Y=530 (config에서 조정 가능)

### 필요한 에셋

| 파일명 | 설명 |
|--------|------|
| `frozen_food_ramen.png` | 라면 |
| `frozen_food_dumpling.png` | 만두 |
| `frozen_food_kimbap.png` | 김밥 |
| `frozen_food_burger.png` | 햄버거 |
| `frozen_food_chicken.png` | 치킨 |
| `frozen_food_pizza.png` | 피자 |
| `frozen_food_tteokbokki.png` | 떡볶이 |

### Config 설정 (`config.py`)

```python
FROZEN_FOOD_SIZE = 40       # 아이콘 크기 (px)
FROZEN_FOOD_GAP = 10        # 아이템 간 간격 (px)
FROZEN_FOOD_TABLE_X = 500   # 테이블 중앙 X 위치
FROZEN_FOOD_TABLE_Y = 530   # 테이블 위 Y 위치
FROZEN_FOOD_MIN_COUNT = 1   # 최소 표시 개수
FROZEN_FOOD_MAX_COUNT = 5   # 최대 표시 개수
```

### 단일 프레임 상세

```
┌─────────────────────────────────┐
│         220px x 393px           │
│     (카드 150x253 + 여백 35px)   │
├─────────────────────────────────┤
│                                 │
│   🔥  🔥  🔥  🔥  🔥  🔥        │ ← 상단 불꽃 (여백 35px)
│  🔥 ┌───────────────┐ 🔥       │
│     │               │          │
│  🔥 │   카드 영역    │ 🔥       │ ← 카드 크기: 150x253
│     │   (투명)      │          │
│     │               │          │
│  🔥 │               │ 🔥       │
│     └───────────────┘          │
│   🔥  🔥  🔥  🔥  🔥  🔥        │ ← 하단 불꽃 (여백 35px)
│                                 │
└─────────────────────────────────┘
         중앙: 카드를 중심으로 불꽃 배치
```

### 규격

| 항목 | 값 |
|------|-----|
| **시트 크기** | 1320px × 393px |
| **프레임 수** | 6 |
| **프레임 크기** | 220px × 393px |
| **카드 영역** | 150px × 253px (중앙, 투명) |
| **불꽃 여백** | 35px (상하좌우) |
| **형식** | PNG (투명 배경) |
| **FPS** | 10 (100ms/frame) |

### 필요한 에셋

| 파일명 | 위험도 | 색상 |
|--------|--------|------|
| `aura_fire_3.png` | 3 (주의) | 주황색 🟠 |
| `aura_fire_5.png` | 5 (위험) | 빨간색 🔴 |
| `aura_fire_7.png` | 7 (매우위험) | 보라색 🟣 |

### 색상 가이드

| 위험도 | 메인 색상 | 보조 색상 (그라데이션) |
|--------|----------|---------------------|
| 3 | `#FFA500` (255, 165, 0) | `#FFD700` (255, 215, 0) |
| 5 | `#FF3232` (255, 50, 50) | `#FF6B6B` (255, 107, 107) |
| 7 | `#643296` (100, 50, 150) | `#9B59B6` (155, 89, 182) |

### 애니메이션 가이드

- 불꽃은 카드 가장자리를 따라 **타오르는 느낌**으로
- 프레임 간 불꽃 위치가 **약간씩 이동** (3-5px)
- 불꽃 크기 **맥동 효과** (10-15% 변화)
- 중앙 카드 영역은 **반드시 투명** (카드가 위에 그려짐)
